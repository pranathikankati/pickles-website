# Amma's Pickles 🫙

A responsive single-page business website for Amma's Pickles —
a homemade pickle brand based in Nalgonda, Telangana.

## Live Demo
🌐 [View Website](https://charming-seahorse-541dbf.netlify.app/)

## Pages / Sections
- **Hero** — Tagline, stats (50+ years recipe, 6 varieties, 100% natural)
- **About** — Brand story and values
- **Products** — 6 pickle varieties with prices
- **Order Form** — Full order form with validation

## Features
- Responsive design — mobile and desktop
- Smooth scroll navigation with anchor links
- Order form with client-side validation
- Success message on form submission
- Product catalogue with prices

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## How to Run
Open `index.html` in any browser. No installation needed.
